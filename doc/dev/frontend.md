```bash
  npm install -g nodeppt

  nodeppt start -p port
```
  # or

```bash
  nodeppt start -p port -d path/for/ppts

  # 支持markdown语法快速创建网页幻灯片。
  nodeppt create ppt-name

  # html版本，
  nodeppt create ppt-name.html
```

  speaker: 演讲者名字

  url: 可以设置链接

  transition: 转场效果，例如：zoomin
